# MARK AI Landing Page

עמוד נחיתה אחד בעברית עם:
- Next.js
- React
- TypeScript
- Tailwind CSS
- RTL מלא
- עיצוב עתידני/טכנולוגי

## הרצה
```bash
npm install
npm run dev
```

ולאחר מכן לפתוח:
```bash
http://localhost:3000
```
